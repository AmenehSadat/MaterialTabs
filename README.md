### Dependency

#### Gradle

Step 1. Add `mavenCentral()` to the `repositories` section of your `build.gradle` file

```
	repositories {
	    mavenCentral()
	}
```

Step 2. Add the MaterialTabs dependency as well

```
	dependencies {
	    // Material Tabs for tab bar.
	    compile 'io.karim:materialtabs:2.0.0
	}
```
